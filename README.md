Download from zip file and add onto your Google Chrome browser. If you never installed an extension before, I have a simple guide with images below.

Steps:

1. Download the zip file

2. Extract the contents of the zip file to a location where you can find it again

3. Open Google Chrome

4. Click on the three dots at the top right of the corner
<img width="1908" height="967" alt="image" src="https://github.com/user-attachments/assets/12182c40-4567-4e17-b62f-8c9ab6864efe" />

5. Hit manage extensions 
<img width="1919" height="1015" alt="image" src="https://github.com/user-attachments/assets/0d247610-8a52-4960-92e8-5c4043f07850" />

6. Enable Developer Mode (toggle all the way at the right)
<img width="1919" height="279" alt="image" src="https://github.com/user-attachments/assets/64edf11e-0af1-4c54-8d39-11137f2385af" />

7. Click Load Unpacked 
<img width="570" height="270" alt="image" src="https://github.com/user-attachments/assets/192e15b2-8ccf-43c1-a529-8fec7b0b0b07" />

8. Select Folder, refresh your browser, and go use it! 
<img width="952" height="593" alt="image" src="https://github.com/user-attachments/assets/b2e10ad9-b5a5-4aae-bbd4-2e30fbd40121" />
